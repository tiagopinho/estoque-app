# 🚀 Início Rápido em 5 Minutos

## 1️⃣ Gerar Token GitHub (2 min)

- Acesse: https://github.com/settings/tokens
- Clique: "Generate new token" → "Generate new token (classic)"
- Selecione: `repo` ✓
- Copie o token (aparece apenas uma vez!)

## 2️⃣ Criar Repositório (1 min)

- Acesse: https://github.com/new
- Nome: `estoque-app`
- Visibilidade: **Public** ⚠️
- Crie o repositório

## 3️⃣ Upload dos Arquivos (1 min)

- Vá para seu repositório
- Clique: "Upload files"
- Arraste todos os arquivos do projeto
- Commit

## 4️⃣ Ativar GitHub Pages (1 min)

- Settings → Pages
- Source: "Deploy from a branch"
- Branch: `main` / Folder: `/`
- Save

## 5️⃣ Configurar Aplicação (1 min)

- Acesse: https://seu-usuario.github.io/estoque-app
- Menu (⚙️) → Configurações
- Preencha:
  - Token: (cole aqui)
  - Usuário: seu_usuario
  - Repositório: estoque-app
- Testar Conexão ✓

**Pronto! 🎉**
